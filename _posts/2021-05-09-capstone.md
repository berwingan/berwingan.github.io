I am glad to announce the submission of my capstone in computational geometry with Professor Saurabh Ray titled "Approximations for Covering in a Streaming Setting, Covering with Orthants"

We present approximations schemes for local variations on the unit disc covering problem in a streaming setting. We give a 2-approximation algorithm to find a hitting set of size 2 when there exists a hitting set of size 1 for the half-space variation. After that, we give a 1.5-approximation algorithm for the dual problem of the interval covering problem when the solutions size is 2. For the lower bound, we show a 1.5 lower bound for the square variation and a 2 lower bound for the half-space variation. For covering with orthants, we first give a construction method to solve the uniform single connection with one type of orthant problem. We then show that using only one type of orthant is equivalent to the planar graph. Furthermore, we worked towards showing arbitrarily large size cliques cannot be made using exactly zero and one intermediary points.
